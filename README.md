# MyDiary – Personal Diary Web Application

MyDiary is a **web-based personal diary application** that lets users write and store their daily thoughts online.
It provides a simple and private space where users can **record daily experiences, manage notes, and look back at past entries**.

The application includes **user authentication**, so each person's diary is private and only accessible to them.

---

## Overview

MyDiary helps users:

* Write and save **daily journal entries**
* View and manage **past diary notes**
* Keep their entries **secure with login authentication**

The goal of the project is to create a **simple, secure, and easy-to-use digital diary**.

---

## Features

### User Authentication

* Secure **signup and login**
* Each user can only access their own diary entries

### Diary Entries

* Write and save **daily notes**
* Edit or manage previous entries
* Store entries safely in the database

### Personal Record Keeping

* Maintain a **history of diary entries**
* Easily revisit past thoughts and experiences
